    • Converting PP-215 Flip Clock timer to 230v

      PP-215 is a beautiful 70s Japan Made flip clock timer used to power up electronics (e.g. audio equipment) at the same time.

      In this project I will convert the Japan model to work on 230v outlets using 3D printing, Arduino Nano, RTC module and a 28BYJ48 stepper motor.

      I created that 3D model of the clock mechanism on the right using RealityScan and used it to create the motor holder model.

    • Vinyl Record Collection Manager

      A command-line interface (CLI) program that allows easy access and management of personal vinyl record collection. Utilizes the Discogs API to retrieve information about records and allows search by query. User can also use commands to perform actions such as viewing, sorting, deleting records, and creating custom records. Convenient tool for vinyl collectors who want to keep track of their collection and access information quickly and easily from the command line.
